📌 About the Project

Renson EcoPark Philippines is a nature-inspired eco-tourism and relaxation destination located in Lopez, Quezon. The website serves as the digital hub for bookings, promotions, and information about the resort, farm, and eco-park experience.

It promotes:

🌿 Nature-based tourism
🏡 Farm & resort accommodations
🎉 Events & celebrations (weddings, birthdays, reunions)
🌊 Relaxation and eco-friendly outdoor experience
🚀 Features of the Website
📅 Online booking system (reservation form)
📍 Google Maps location integration
🖼️ Gallery of farm & eco-park views
💬 Contact & inquiry section
📱 Mobile responsive design
🎯 Promo pages for events & discounts
🌐 Simple fast-loading HTML structure
